# Governance Themes v1.0

## Purpose

This document identifies the recurring governance themes that appear across portfolio projects, research, publications and external validation.

The goal is not to create new concepts.

The goal is to identify concepts that repeatedly emerged through evidence.

---

# Theme 1 — Authority Preservation

## Core Question

Who retains decision authority?

## Recurring Position

Evaluation, recommendation and analysis must remain separated from authority.

## Evidence Sources

* IPC1 — Authority Failure
* IPC3 — Externalized Governance
* ARC
* Governance Boundary Research
* Oleksandr discussions

---

# Theme 2 — Admissibility

## Core Question

Can a decision be safely executed?

## Recurring Position

Validity alone is insufficient.

A decision must also be admissible.

## Evidence Sources

* IPC5
* ARC
* Kari discussions
* Ricardo discussions

---

# Theme 3 — Governance Observability

## Core Question

Can governance state be reconstructed?

## Recurring Position

Governance must remain observable, auditable and replayable.

## Evidence Sources

* CEIE
* Runtime Reviews
* Audit Architecture
* Emanuel discussions

# Theme 4 — Evidence-Based Governance

## Core Question

How do we know governance is functioning?

## Recurring Position

Declared controls are insufficient.

Operational evidence must exist.

## Evidence Sources

* IPC9
* CEIE Pressure Program
* Piotr discussions
